# Kraven Financial Performance & Margin Risk Assessment (Excel Project)

## Project Overview
This project analyzes Kraven’s 2013–2014 financial performance using Microsoft Excel exclusively. The objective was to evaluate revenue trends, validate profitability calculations, diagnose margin risks, and forecast short-term revenue outlook.

---

## Business Problem

Leadership required clarity on:

- True profitability drivers
- Segment-level efficiency
- Margin compression from discounting
- Loss-making segments and products
- Revenue sustainability and forward outlook

---

## Dataset

Source: Microsoft Power BI Financial Sample Dataset  
Period Covered: 2013 (partial) – 2014 (full year)

Columns Included:
Segment, Country, Product, Discount Band, Units Sold, Manufacturing Price, Sale Price, Gross Sales, Discounts, Sales, COGS, Profit, Date, Month, Year.

---

## Tools & Techniques Used

- Microsoft Excel
- Pivot Tables

- Financial Recalculation Validation
- Margin Binning
- Discount Impact Analysis
- FORECAST.ETS
- Excel Forecast Sheet
- Executive Dashboard Design

---

## Key Analysis Areas

### 1. Revenue Performance
- Monthly trend analysis
- Q4 seasonal peak identification
- Regional revenue contribution

### 2. Profitability Diagnostics
- Weighted segment margin calculation
- High vs low margin product identification
- Loss-making segment detection
- Volume vs profitability assessment

### 3. Discount Risk Analysis
- Margin compression across discount bands
- Profit concentration impact

### 4. Forecasting
- 6-month revenue forecast using ETS modeling
- Seasonality detection (12-month pattern)
- Projected -2.6% normalization from December 2014 peak

---

## Key Insights

- 67.41% of total profit comes from Government segment (dependency risk).
- Enterprise segment operates at a loss (-3.1% margin).
- High discount levels compress margin efficiency.
- Revenue peaked in Q4 2014.
- Forecast indicates slight 6-month normalization (-2.6%) post seasonal high.
